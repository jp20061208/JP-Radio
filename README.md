# JP-Radio 本地音乐播放器 

## 功能特点
- 🎵 支持本地音乐播放 + LRC 歌词同步显示
- 🖥️ 托盘常驻，后台不中断播放
- 📦 免安装绿色版，解压即用
- 🎛️ 支持进度控制、音量调节、播放列表

## 使用方法
1. 下载 `JP-Radio-v0.1.0.zip` 并解压
2. 运行 `JP-Radio.exe`
3. 选择你的音乐文件夹（包含 `Music` 和 `Lyrics` 子目录）
